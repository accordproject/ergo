# The Emacs mode for Ergo

Install this mode by placing `ergo-mode.el` in a directory of your choice, and adding the following lines to your `emacs.d` configuration:

```
(add-to-list 'load-path "/place/where/you/put/it/")
(require 'ergo-mode)
```
